## Versionning

Tout est sauvegardé, sécurisé, versionné.

**Convention de nommage :**
- Format : `YYYY-MM-DD_NomDocument.ext`
- Exemple : `2026-01-26_Protocole.pdf`

**Anciennes versions :**
- Dans un dossier `_NomDocument/` à côté du fichier courant
- Exemple : `_Protocole/` contient les versions précédentes

**Exemple :**
```
Projet Achat Maison/
├── 2026-01-20_Compromis.pdf
└── _Compromis/
    └── 2026-01-10_Compromis.pdf
```
