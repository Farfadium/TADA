### Revue projets

**Déclencheurs :**

| Type | Valeur |
|------|--------|
| Temps | Vendredi |
| Tag | #revue |

**Impact :** Contenu

**Contexte :**
Faire le point hebdomadaire sur tous les projets actifs pour identifier les blocages et planifier la semaine suivante.

**Actions :**

1. Lister les projets dans `NOW/`
2. Pour chaque projet, lire `index.md` et extraire :
   - Statut actuel
   - Dernière activité
   - Tâches en cours / bloquées
3. Présenter un résumé global
4. Proposer des actions pour la semaine suivante
5. Mettre à jour les index si nécessaire

**Validation requise :** Oui (pour les mises à jour)

**Format de présentation :**
```
## Revue du [date]

### [Projet 1]
- **Statut :** En cours / Bloqué / En attente
- **Dernière activité :** [date]
- **À faire :** ...
- **Blocages :** ...

### [Projet 2]
...

---

**Actions proposées pour la semaine :**
1. ...
2. ...
```
